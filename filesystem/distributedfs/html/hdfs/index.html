Hadoop Distributed File System

1. Intro

1.1. Overview

HDFS是一个分布式文件系统，建立在廉价的服务器设备之上。HDFS是对应与GFS(Google File System)的开源实现，GFS开源实现的另外一个版本是Moose File System。HDFS通过连接多个不同计算机设备构成了一个巨大的存储容量空间。HDFS由Client、NameNode、DataNode三部分组成。

1.2. Installation and Use

安装使用分为standlone和distributed，具体安装见Hadoop Document。以后详细的参数优化需要查看具体的每个参数的含义，每个参数的含义见hdfs-default.html

2. Features

  • Federation
  • 数据节点Replication
  • High Available方法
  • Central Cache（集中化缓存）
  • Tier分层存储技术
  • SSD技术支持
  • Block Pool技术
